
■ 1. OpenAI API 初期化

・dotenv で API キーを環境変数から読み込み

・OpenAI クライアントを生成して GPT 呼び出しを準備


■ 2. 読み仮名チェック関数（check_readings）

・人名リストを改行結合してプロンプトを構築

・「名前,カタカナ読み」の形式で出力するよう GPT に指示

・gpt-4o-mini を使用し誤読・誤表記を自動修正

・返却内容を strip して整形


■ 3. 日本語人名ファイル読み込み

・人物名が保存されたファイル.txt を UTF-8 で読み込み

・空行を除いた名前リストを作成


■ 4. GPT による読み仮名検証・補正

・読み込んだ全名前を check_readings() に渡す

・GPT によりカタカナ読みへ修正された結果を取得


■ 5. 修正結果の保存処理

・同じファイルに上書き保存（追記ではなく完全置換）

・整形済みの「名前,読み」をそのまま保存


■ 6. 実行ログ

・読み仮名自動チェック完了メッセージを表示
