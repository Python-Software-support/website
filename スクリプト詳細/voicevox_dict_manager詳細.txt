
■ 1. 設定値・パス管理

・VOICEVOX エンジン URL の定義

・辞書バックアップ保存先パスの管理

・新語ファイルパスと登録履歴ファイルの管理


■ 2. 登録ログの読み書き

・registered_words.json を読み込みセット化

・登録済み単語を小文字化して重複を防止

・更新内容を JSON 形式で保存


■ 3. 辞書バックアップ処理（backup_dict）

・/user_dict を取得して JSON をバックアップ保存

・エラー発生時は例外処理で通知

・成功時は user_dict_backup.json を生成・更新


■ 4. 辞書復元処理（restore_dict）

・既存 user_dict.json を安全に削除

・バックアップが無ければ警告を表示

・バックアップ内容を POST して辞書を復元


■ 5. 新語登録処理（add_new_words）

・VOICEVOX の既存辞書内容を取得

・ローカル履歴と照合して未登録語だけを抽出

・surface / pron / accent を指定して登録 API に送信

・登録済み語・発音不足・スペース含みはスキップ

・成功時に履歴ファイルを更新して保存


■ 6. 全単語削除処理（delete_all_words）

・辞書エントリを UUID ごとに取得

・DELETE リクエストを用いて個別削除

・成功 / 失敗をログ出力

・完了後に登録履歴ファイルも削除


■ 7. メニュー UI（コンソール操作）

・新語登録 / バックアップ / 復元 / 全削除 / 終了 を選択方式で表示

・TXT ファイル一覧を番号付きで提示

・選択番号に応じて機能を実行


■ 8. メイン処理ループ

・while True でメニューを連続表示

・ユーザー入力で動作を振り分け

・無効入力時は警告を表示して継続
